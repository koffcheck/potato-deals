# Potato Deals vX.Y.Z

## Download

Install using `potato-deals-vX.Y.Z.zip` from the `Assets` section.

Do not use:

- `Source code (zip)`
- `Source code (tar.gz)`

Those files are GitHub source archives and cannot be installed in Decky Loader.

## Installation

1. Open Decky Loader.
2. Go to `Settings -> Developer`.
3. Select `Install from ZIP`.
4. Choose `potato-deals-vX.Y.Z.zip`.
